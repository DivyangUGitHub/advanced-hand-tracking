# Advanced Hand Tracking & Gesture Recognition

Real-time hand tracking and gesture recognition using MediaPipe and OpenCV.

## Features
- Real-time hand landmark detection
- 8+ gesture recognition with smoothing
- Robust hand tracking with confidence scoring
- Visual feedback with landmarks and gesture labels
